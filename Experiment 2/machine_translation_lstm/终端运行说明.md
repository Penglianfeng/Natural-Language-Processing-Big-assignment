# 终端运行说明

在当前工作区根目录运行：

```bash
python machine_translation_lstm\seq2seq_lstm_attention.py
```

程序会自动完成：

1. 构造英中平行语料；
2. 训练普通多层 LSTM Seq2Seq 模型；
3. 训练带注意力机制的多层 LSTM Seq2Seq 模型；
4. 输出样例翻译结果；
5. 保存 loss 曲线、注意力热力图和实验结果截图。

运行结果保存在：

```text
machine_translation_lstm\results\
```

主要文件：

- `result_screenshot.png`：可直接粘贴到实验报告的运行结果截图；
- `loss_curve.png`：训练 loss 曲线；
- `attention_heatmap.png`：注意力机制热力图；
- `predictions.txt`：终端输出式的翻译结果；
- `metrics.json`：实验指标。

本实验不需要 Jupyter notebook 内核。若老师要求“Jupyter notebook 或 PyCharm”，本项目选择 PyCharm/终端脚本方式完成。
